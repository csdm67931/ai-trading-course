## Gemini Gem 建立步驟（快速參考）

1. 前往 https://gemini.google.com，確認右上角是學校 Google 帳號
2. 左側選單找到「Gems」→ 點「新建 Gem」
3. 名稱填入：`我的交易 AI 助理`
4. 說明欄位：開啟 `預載Prompt.txt`，將全部內容複製貼入
5. 儲存，從這個 Gem 開啟新對話測試

---

## Github repo 建立步驟（快速參考）

1. 前往 https://github.com → 右上角「Sign up」申辦帳號
2. 登入後點「+」→「New repository」
3. Repository name 填入：`ai-trading-journal`
4. 選 **Public**，勾選「Add a README file」
5. 點「Create repository」
6. 進入 repo，點鉛筆圖示編輯 README.md
7. 將 `我的交易日誌範本` 的內容貼入，修改個人資訊後儲存

---

## 我的交易日誌範本

建立 repo 後，將以下內容貼入你的 README.md：

```markdown
# AI 交易日誌

姓名：（填入你的名字）
學校：（填入你的學校）
課程：AI 輔助交易工具課程

## 我的 AI 交易主題

（說明你想讓 AI 幫你在交易上做什麼事，一到兩句話即可）

範例：
- 我想讓 AI 每天早上自動摘要台指期的重要觀察點
- 我想讓 AI 根據新聞情緒給我進出場方向的建議
- 我想讓 AI 幫我記錄每筆交易並自動分析原因

## 第一次 AI 對話紀錄

日期：（填入今天日期）

問題：（你問 Gemini 的問題）

AI 回答摘要：（AI 回答的重點，3 到 5 行）

我的觀察：（這個回答有用嗎？哪裡可以改進？）
```

## 常見問題

**Q：找不到 Gemini 左側的「Gems」選單？**
確認你登入的是學校 Google 帳號（有 Gemini Advanced 權限），個人免費帳號沒有 Gems 功能。

**Q：Github 申辦需要信用卡嗎？**
不需要，免費帳號可以直接申辦使用。

**Q：Antigravity 跟不上沒關係嗎？**
完全沒關係，第一堂課的 Antigravity 以觀念為主，第二和第三堂會有更多時間練習。
